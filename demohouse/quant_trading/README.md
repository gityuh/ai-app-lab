# 股票交易 AI 辅助预测工具

股票量化交易是一个挺个性化的软件，每个人有自己的买卖策略。市面上的量化交易软件，要么需要收费，要么有较高使用门槛。所以，我们想利用 AI 做一款开源的股票量化交易系统，**让每个人都能用上量化交易，并且每个人都能定制自己的交易策略**。
